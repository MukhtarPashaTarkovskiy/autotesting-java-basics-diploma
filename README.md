# Autotesting Java Basics Diploma

Дипломный проект по автоматизированному тестированию веб-приложения на Java.

## Описание проекта

Проект содержит UI-автотесты для учебного интернет-магазина Skillbox.

Тестируемое приложение:

```text
http://intershop5.skillbox.ru/
```

Автотесты проверяют основные пользовательские сценарии: переходы по разделам сайта, работу каталога, добавление товаров в корзину, изменение количества товаров, применение купона, оформление заказа, регистрацию, авторизацию и восстановление пароля.

## Технологии

- Java 17
- Selenium WebDriver
- JUnit 5
- Maven
- WebDriverManager
- XPath
- CSS selectors
- Git

## Что проверяется

### Главная страница

- Переход в раздел «Книги»
- Переход в раздел «Планшеты»
- Переход в раздел «Фото/видео»
- Переход на страницу товара из промоблока
- Переход на страницу авторизации через кнопку «Войти»

### Каталог товаров

- Поиск товара через строку поиска
- Переход в каталог через меню хедера
- Переход в каталог через футер
- Добавление товара в корзину из каталога
- Переход в карточку товара

### Корзина

- Добавление товара в корзину
- Изменение количества товара в корзине
- Удаление товара из корзины
- Восстановление удалённого товара
- Применение купона на скидку
- Переход к оформлению заказа

### Оформление заказа

- Заполнение формы оформления заказа
- Оформление заказа с оплатой при получении
- Применение купона на странице оформления заказа
- Проверка валидации обязательных полей

### Регистрация и авторизация

- Регистрация нового пользователя
- Авторизация пользователя
- Восстановление пароля

## Как запустить тесты

### 1. Клонировать репозиторий

```bash
git clone https://github.com/MukhtarPashaTarkovskiy/autotesting-java-basics-diploma.git
```

### 2. Перейти в папку Maven-проекта

```bash
cd autotesting-java-basics-diploma/Diploma
```

### 3. Запустить все тесты

```bash
mvn test
```

### 4. Запустить отдельный тестовый класс

```bash
mvn -Dtest=CartPageTests test
```

Примеры:

```bash
mvn -Dtest=HomePageTests test
mvn -Dtest=ProductCatalogTests test
mvn -Dtest=RegistrationAndAuthorizationTests test
```

### 5. Запустить отдельный тест

```bash
mvn -Dtest=CartPageTests#testAdding5ItemsToCart test
```

## Особенности проекта

- Тестируемый сайт не входит в репозиторий, так как это внешний учебный сайт.
- ChromeDriver управляется автоматически через WebDriverManager.
- В проекте используется Selenium WebDriver и явные ожидания `WebDriverWait`.
- Часть тестов зависит от текущего состояния внешнего учебного сайта и его разметки.
- Некоторые проверки, связанные с товаром `Lada` в слайдере главной страницы, могут быть временно отключены через `@Disabled`, так как слайдер и карточка товара отображаются нестабильно.
